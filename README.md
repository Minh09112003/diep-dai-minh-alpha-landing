# Diệp Đại Minh - Alpha Track Landing Page

## Tổng quan dự án
Đây là bản **clone pixel-perfect** của trang landing page sản phẩm **Lymph CC Select** từ Sculptique.  
Trang này được xây dựng cho **bài kiểm tra Practical Test của Alpha Track e-Commerce Developer tại PATI Group**.

**Trang tham khảo:** [https://trysculptique.com/products/lymph-cc-select](https://trysculptique.com/products/lymph-cc-select)

## Công nghệ sử dụng
- HTML5
- CSS3 (Sử dụng Tailwind CSS)
- JavaScript (cho slider, popup hoặc các tương tác nếu có)
- Responsive: Desktop, Tablet, Mobile (test trên iPhone 14 Pro Max)

## Tính năng
- Layout chính xác tuyệt đối so với trang gốc (spacing, typography, màu sắc, font)
- Hoàn toàn responsive trên nhiều thiết bị
- Tương tác UI: hover, slider, popup (nếu có)
- Code sạch, dễ đọc và có cấu trúc rõ ràng

## Hướng dẫn chạy

### 1. Xem online (Live demo)
- Bản live đã deploy trên Vercel:  
[https://diep-dai-minh-alpha-landing-page.vercel.app/](https://diep-dai-minh-alpha-landing-page.vercel.app/)

### 2. Mở offline bằng VS Code + Live Server
1. Clone repository về máy:
git clone https://github.com/DiepDaiMinh/diep-dai-minh-alpha-landing.git
2. Mở thư mục project bằng VS Code, cài extension Live Server (nếu chưa có), rồi click “Go Live”.
Trình duyệt sẽ mở localhost và tất cả ảnh, CSS, JS sẽ load đầy đủ.
⚠️ Lưu ý: nếu mở trực tiếp file index.html bằng double click, một số ảnh hoặc JS có thể không load đúng do đường dẫn relative.
